# Servidor

Comandos para iniciar o site do zero:

```sh
# Ter o Nodejs instalado
# Instalação dos pacotes
    npm i 
# Configurar o .env.local
# Instalação da base de dados com  a tabela
    npm run migrate
# Criação da seed (opcional)
    npm run seed
# Build do next
    npm run build
# Apenas para teste
    npm start
```
